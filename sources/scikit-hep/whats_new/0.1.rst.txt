What's new in 0.1
.................

- Initial release.
- Package contains several modules (at different stages of development):

  - Units and constants typical in Particle Physics, in the HEP System of Units.
  - Utilities around the Particle Data Group table and IDs.
  - Math classes and functions handy for kinematics.
  - Basic dataset-like classes for interoperability, including data provenance information.
  - Dynamic programming algorithm for finding the optimal adaptive-width histogram.
